# Sistema de Atendimento Carol · Taís Madureira Makeup

Scripts de atendimento WhatsApp organizados por público, com tabela de serviços e botão de cópia rápida.

## Como usar

Abra o arquivo `index.html` diretamente no navegador — ou acesse pelo link do GitHub Pages após ativar a hospedagem.

## Publicar no GitHub Pages

1. Suba este repositório no GitHub
2. Vá em **Settings → Pages**
3. Em *Branch*, selecione `main` e pasta `/ (root)`
4. Clique em **Save**
5. Aguarde ~1 minuto e acesse o link gerado
